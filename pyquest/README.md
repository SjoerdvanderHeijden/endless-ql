## Dependencies
### Python
Python version 3 or higher.

#### Packages
- PLY
- PyQt5
- termcolor
- multimethods

## Setup
```
sudo pip3 install ply
sudo pip3 install pyqt5
sudo pip3 install termcolor
sudo pip3 install multimethods
```


## Running
```
python GUI/app.py
```
